"# Presentación del Proyecto Iglesia Jerusalén Web Page

## Introducción
La página web de Iglesia Jerusalén es un proyecto que busca proporcionar una plataforma en línea para la comunidad de la iglesia.

## Características y Funciones
* Gestión de usuarios y roles
* Panel de administración
* Vista pública
* Tienda en línea
* Integración con GitHub, Vercel, Cloudinary y Supabase

## Herramientas y Tecnologías
* Frontend: HTML, CSS, JavaScript
* Backend: Supabase
* Despliegue: Vercel
* Almacenamiento de imágenes: Cloudinary

## Dashboard y Panel Admin
* Visión general del sitio web
* Gestión de usuarios y roles
* Configuración del sitio web

## Vista Pública
* Página de inicio
* Sección de noticias
* Sección de eventos

## Tienda en Línea
* Catálogo de productos
* Carrito de compras
* Pagos en línea

## Detalles Técnicos
* Base de datos: Supabase
* Servidor: Vercel
* Almacenamiento de imágenes: Cloudinary
* Integración con GitHub para versionamiento

## Conclusión
La página web de Iglesia Jerusalén es un proyecto complejo que busca proporcionar una plataforma en línea para la comunidad de la iglesia. Con sus características y funciones, es una herramienta valiosa para la gestión y el crecimiento de la comunidad.
"
